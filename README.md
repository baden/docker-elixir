# docker-elixir

My base Elixir docker image.

## Build local image

```
  make build
```

## Run local image

```
  make run
```

## Run image from [DockerHUB](https://hub.docker.com/r/baden/elixir/)

No need clone repository. You can use this method from anywhere.

```
  docker run -it --rm baden/elixir
```
